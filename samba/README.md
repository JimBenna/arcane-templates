**SMB server in a container**

This is the implementation of [samba](https://www.samba.org). The purpose is to easyly setting up some network shares.

The container has been build by [dockur](https://github.com/dockur/samba) and it is pulled from [Docker Hub](https://hub.docker.com/r/dockurr/samba)

Please read the how-to on their github page to setup this container properly.
