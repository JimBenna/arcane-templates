**Qemu in a container to access it from a browser**

This enables to host a linux system, or any other supported operation system by Qemu.

The container has been build by [QEMU Tools](https://github.com/qemus) and it is pulled from [Docker Hub](https://hub.docker.com/r/qemux/qemu)

Please read the how-to on their github page or docker hub to setup this container properly.
