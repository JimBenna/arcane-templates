**FileStash.**


Enterprise File Management Software

The Open Source Platform for File Sharing / Document Management / MFT / Archiving

- Secure File Sharing
- SSO Integration & Authorisation
- Gateway for other file storage services (FTP, SFTP, WebDAV SMB, LDAP, S3,)
- etc...

The container has been build by the author of the app [Mickael Kerjean](https://github.com/mickael-kerjean) and it is pulled from [Docker Hub](https://hub.docker.com/r/machines/filestash)

Deployment how-to and information on the application website : https://www.filestash.app/docs/.
