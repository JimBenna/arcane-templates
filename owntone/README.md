**Owntone is an audio media server**

The impressive list of implemented features is listed in its github page : [Owntone Website](https://owntone.github.io/owntone-server/)

But here is a short list of those features :
- Stream to  AirPlay (synchronized multiroom) and  Chromecast devices
- Spotify support
- Remote App control (native IOS apple remote app)
