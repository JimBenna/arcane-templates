**VsCodium in a web browser**

[Vscodium](https://vscodium.com) is a community-driven, freely-licensed binary distribution of the remote host web component of Microsoft's editor VS Code.

The container has been build by [linuxserver.io](https://docs.linuxserver.io/images/docker-vscodium-web/) and it is pulled from [Docker Hub](https://hub.docker.com/r/linuxserver/vscodium-web)

Ok, you are going to tell me "Why this version ?" From my point of view if you want to copy paste data from your desktop to this application it's more convenient to use it with a native browser.

The data can be defined with a network samba share, so all is used stored directly from the network, nothing on your workstation :computer:
